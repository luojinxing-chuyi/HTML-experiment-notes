    a name="" herf="" target=""

HTML列表

HTML列表支持有序、无序和自定义。

  1. 无序列表是一个项目列表，此列项目使用粗体原点（典型的小黑圆圈）进行标记。 
    <ul><li>列表内容</li></ul>
    practice:

    使用type(类型)属性给列表设置属性值，更改粗体原点标记。

    1. disc:● 
      <ul type="disc">
      practice:
    2. circle:○ 
      <ul type="circle"
      practice:
    3. square:■ 
      <ul type="square"
      practice:
  2. 有序列表同样也是一个列表项目，项目使用数字进行标记。 
    <ol><li>列表内容</li></ol>
    practice:

    使用type(类型）属性给列表设置属性值，更改数字标记。

    1. 大写字母列表 
      <ol type="A">
      practice:
    2. 小写字母列表 
      <ol type="a">
      practice:
    3. 大写罗马字母列表 
      <ol type="Ⅰ">
      practice:
    4. 小写罗马字母列表 
      <ol type="ⅰ">
      practice:
  3. 自定义列表不仅仅是一列项目，而是项目及其注释的组合。由<dl>定义自定义列表、<dt>定义项目、<dd>定义项目描述。 
    <dl><dt>项目内容</dt><dd>项目描述</dd></dl>

HTML表单

HTML 表单用于搜集不同类型的用户输入。

  1. <form>元素定义 HTML 表单 
    <form>其他表单元素</form>
    practice:
    1. action属性定义在提交表单时的执行的动作。指定了某个服务器脚本来处理被提交表单。 
      <form action="action_page.php">
      practice:
    2. method属性规定在提交表单时所用的HTTP方法:GET或POST。 
      <form action="action_page.php" method="GET">or method="POST"
      practice:
    3. accept-charset属性规定服务器用哪种字符集处理表单数据。ISO-8859-1 - 拉丁字母表的字符编码/gb2312 - 简体中文字符集。 
      [form action="action_form.asp" accept-charset="ISO-8859-1"]
      practice:
    4. autocomplete属性规定表单是否启动自动完成功能。也可以独立使用在input元素中。 
      [form action="action_form.asp" method="get" autocomplete="on/off"]
      practice:
    5. enctype属性规定在发送到服务器之前应该如何对表单数据进行编码。 
      [form action="form_action.asp" enctype="text/plain"]
      practice:
    6. name 属性规定表单的名称。form 元素的 name 属性提供了一种在脚本中引用表单的方法。 
      [form action="form_action.asp" method="get" name="myForm"]
      practice:
    7. novalidate 属性规定当提交表单时不对其进行验证。如果使用该属性，则表单不会验证表单的输入。注释:novalidate 属性适用于：
      ，以及以下类型的 标签：text, search, url, telephone, email, password, date pickers, range 以及 color。 
      [form action="demo_form.asp" novalidate="novalidate"]
      practice:
  2. <input>是最重要的表单元素。根据type属性有很多不同的形态。
    1. 属性值:text定义用于文本输入的导航输入字段。 
      <input type="text">
      practice:
    2. 属性值:radio定义单选按钮。 
      <input type="radio">
      practice:
    3. 属性值:submit向表单处理程序提交表单的按钮。 
      <input type="submit">
      practice:
    4. 给每个输入字段设置一个name属性，确保正确的提交。 
      <input type="text" name="firstnanm" value="luo">
      practice:
  3. <fieldset>元素组合表单中的相关数据，也可以理解为容器。
    <legend>元素可以给<fieldset>元素定义一个标题。 
    <fieldset><legend>填写个人信息</legend></fieldset>
    practice:
